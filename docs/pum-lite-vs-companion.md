# PUM Lite vs PUM Companion — Feature Parity

This page compares **PUM Lite** and **PUM Companion** (Full Version) feature by feature. Use it to decide which app fits your play style, or to see what you gain by upgrading.

For a general introduction to PUM Lite, see [PUM Lite](pum-lite.md).

---

## Features in Both Apps

These core features are included in PUM Lite and PUM Companion.

| Feature | PUM Lite | PUM Companion | Notes |
|---------|:--------:|:-------------:|-------|
| Unlimited Stories | Yes | Yes | Play your stories without any story or character amount limits. |
| Random Tables | Yes | Yes | Create unlimited simple and weighted random tables. |
| Image Viewer | Yes | Yes | Lite supports viewing images attached to entities. |
| PUM Plot Tracks | Yes | Yes | Use PUM Plot Tracks to control the pace of your games. |
| Game Creation Wizard | Yes | Yes | The app helps you create games from scratch and brainstorm. |
| Analog Play | Yes | Yes | Play along your own physical dice and handwriting with the app. |
| Episode Continuation | Yes | Yes | Bring to new plot carries over characters/plot tracks to new games. |
| PDF Story Export | Yes | Yes | Export your story journal as a shareable PDF file. |
| Save Game Export | Yes | Yes | Save, share, backup your games in `.pumc` file format. |
| Global Search | Yes | Yes | Quickly find entities and journal text (Ctrl+F / Ctrl+Space). |

---

## Features with Differences

These features exist in both apps, but with different scope or depth.

| Feature | PUM Lite | PUM Companion | Notes |
|---------|:--------:|:-------------:|-------|
| Oracle System | PUM | PUM+SUM+GUM | Play with all the unfolding machines in the Companion version. |
| Look & Feel | 3 Themes | 7+ Themes | Enjoy more variety of styles for the app and any future ones. |
| New Features and updates | Basic | Improving | New features and community suggestions to improve the app. |

---

## PUM Companion Only

These features are not available in PUM Lite. Some require optional DLC on PUM Companion.

| Feature | PUM Lite | PUM Companion | Notes |
|---------|:--------:|:-------------:|-------|
| Advance Tables | No | Yes | Complex nested tables, dice rolling in entries, and multi-call tables. |
| Multi-Vault Support | No | Yes | Companion supports organizing games in separate folders/vaults (Windows, macOS, Linux). |
| Multi-Tab Journals | No | Yes | Companion allows separating backstory/inventory/relationships. |
| Templates and Files | No | Yes | Companion lets you manage reusable characters, tables, and sheets. |
| Entity Interlinking | No | Yes | Companion supports hyperlinking entities together. |
| Entity Classification | No | Yes | Tag your entities to better organize and search through. |
| Entity Plot Nodes | No | Yes | Link plot nodes to Compendiums to expand your World Building. |
| Entity PDF Export | No | Yes | Export any of your entities into a shareable printable PDF file. |
| Table Classification | No | Yes | Organize tables in groups and perform bulk actions. |
| Advanced Table Macros | No | Yes | Enhance random tables with nested tables and in-line dice rolls. |
| PDF Viewer | No | Yes | View PDFs attached to entities with bookmarks/search/text selection. |
| PDF Sheet Conversion | No | Yes | Companion converts PDF pages to editable character sheets and maps. |
| PDF Page Interlinking | No | Yes | Links from journal entries to specific PDF pages for convenience. |
| Voice Notes | No | Yes | Record and play by voice and record your audio notes in the story. |

| Sheets and Maps | No | Yes | Image Editor supports character tokens, adding texts, and drawing. |
| Image and PDF Pinning | No | Yes | Pin images and PDFs into floating convenient shortcuts on screen. |

| Multiple-Plot Tracks | No | Yes | Create different Plot Tracks for your side-quests and threads. |
| Custom Plot Tracks | No | Yes | Customize and manage your Plot Tracks, or operate multiple in parallel. |
| PUM Optional Rules | No | Yes | Timed plot beat events and disrupted dice for PUM. |
| Bulk Entity Management | No | Yes | Companion bulk-creates and export entities and tables from files. |
| Initiative Tracker | No | Yes | Order your characters in a dynamic tracker for talking or fighting. |
| Google Drive/iCloud Sync | No | Yes | Conveniently sync your games between your devices with privacy. |
| Decks & Cards | No | Yes (DLC) | Play with cards and tarot decks in your games. |
| Builder Widgets | No | Yes (DLC) | Add health trackers, progress bars, and interactive widgets. |
| Crystal Theme | No | Yes (DLC) | Customizable Look & Feel and style to adapt to your story. |
| 3D Dice | No | Yes (DLC) | Animated 3D Dice rolling on screen and create your own as well. |
| AI Assistant | No | Yes (DLC) | Boost your stories with AI ideas, oracle interpretations, and chat. |
